BusMaster transfer

BusMaster transfer is a function of PCI bus board that directly input/output from board to user memory no through CPU like DMA.
By using this BusMaster transfer function, the high-speed input/output used the timer inside of board, and so on, can be correctly performed.
BusMaster transfer performs the transfer between the board and memory of PC when the CPU does not use the bus.
Transfer error may occur when load of system heightens or board cannot acquire bus.
