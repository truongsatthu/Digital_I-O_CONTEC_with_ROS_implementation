Mode setting

After acquiring the device ID, mode setting (DioDmSetDirection) to decide the input/output direction should be implemented.
The input/output direction is selected from 32-point input, 32-point output, 16-point input or 16-point output.

Next, the Stand-alone, Master and Slave settings are implemented.

Stand-alone should be set to use one board only, or for each board to be used independently (DioDmSetStandAlone).
When a number of boards are used in synchronization, one of them should be set as the output board (master) (DioDmSetMasterCfg),
and all other boards should be set to receive synchronized signals (Slave) (DioDmSetSlaveCfg).

It is convenient to use a synchronization control connector (SC connector) when using a number of boards synchronously.
For such a case, refer to the chapter on synchronization control connectors.
