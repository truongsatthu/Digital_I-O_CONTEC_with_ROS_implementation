DioDmSetDirection

Function

Sets input/output direction of the specified device.

Format

Ret = DioDmSetDirection(Id, Direction);

Parameters

Id [C:short]
Specifies the device ID retrieved from DioInit.

Directoin [C:unsigned long]
Specify input/output direction.

Define Value Description
PI_32132-bit input
PO_32232-bit output
PIO_1616316-bit input, 16-bit output

Return values

Ret [C:long]

Define Value Description
DIO_ERR_SUCCESS0Normal Complete
DIO_ERR_DLL_INVALID_ID10001Invalid ID is specified.
DIO_ERR_DLL_CALL_DRIVER10002Driver cannot be called (failed in ioctl).
DIO_ERR_SYS_DIRECTION50000I/O direction is outside of the setting range.
The other errors: (See also: Error code details)

Initial value

Initial value of I/O direction is PI_32.

Remarks

Initial value of I/O direction is PI_32.

Example

C       Ret = DioDmSetDirection(Id, PI_32);
