PDIO_DM_STOP_CALLBACK

Function

Transfer stop callback type.

Format

CallBackProc(Id, Message, Dir, Param);

Parameters

Id [C:short]
Specifies the device ID retrieved from DioInit.

Message [C:short]
The message ID that becomes the cause of invoking the callback function is passed.

Define Value
DIO_DMM_STOP1400H

Dir [C:unsigned long]
Transfer direction is passed.

Define Value Description
DIODM_DIR_IN0x01Input
DIODM_DIR_OUT0x02Output

Param [C:void *]
The optional parameter passed through DioDmSetStopEvent is passed.

Return values

None

Remarks

Execute substance definition of transfer stop callback function at this format.

See also

DioDmSetStopEvent
