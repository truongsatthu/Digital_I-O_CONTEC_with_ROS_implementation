Status list

Statuses acquired by the status acquisition function (DioDmGetStatus) are explained.
Statuses are expressed as the logical sums shown below.

Define Value Input/output Types Contents
DIODM_STATUS_BMSTOP00000001hInput/outputInformationIndicates that BusMaster transfer is complete.
DIODM_STATUS_PIOSTART00000002hInput/outputInformationIndicates that PIO input/output has started.
DIODM_STATUS_PIOSTOP00000004hInput/outputInformationIndicates that PIO input/output has stopped.
DIODM_STATUS_TRGIN00000008hInput/outputInformationIndicates that the start signal is inserted by external start.
DIODM_STATUS_OVERRUN00000010hInput/outputWarningIndicates that start signals are inserted twice or more by external start.
Transfer can be continued without difficulty.
Ignore it if there is no problem with the external connection signal specification.
