DioGetPatternEventStatus

Function

The state of the event of the failsafe function is acquired.

Format

Ret = DioGetPatternEventStatus(Id, &Status);

Parameters

Id [C:short]
Specifies the device ID retrieved from DioInit.

Status [C:short *]
Beginning/stop of the event operation is returned.

Value[Dec] Meaning
0The event has not been generated.
1The event was generated.

Return Values

Ret [C:long]

Define Value Description
DIO_ERR_SUCCESS0Normal Complete
DIO_ERR_DLL_INVALID_ID10001Invalid ID is specified.
The other errors: (See also: Error code details)

Initial Values

None

Remarks

The state of the event operation of the failsafe function is acquired.

Example

C       Ret = DioGetPatternEventStatus(Id, &Status);
