Echo Back

Our digital input and output boards have the monitor function for output ports.
Reading the output value is called "echo back".

If the board features the monitor function for output ports, the echo back function return the input value from hardware.
If the board doesn't feature the monitor function, the echo back function return the value stored in the driver work.

Please refer to the device's hardware specification to see whether the device features the monitor function for output port or not.
