DioGet8255Mode

Function

8255 mode setup is acquired per chip.

Format

Ret = DioGet8255Mode(Id, ChipNo, &CtrlWord);

Parameters

Id [C:short]
Specifies the device ID retrieved from DioInit.

ChipNo [C:unsigned short]
Specify a LSIi8255 chip number.

CtrlWord [C:unsigned short]
Specify the address of the variable which stores control word.
The setting value list of control word is shown below.

Initial Values

At the time of system starting, all ports are setup of an input.
Once setting up control word by DioSet8255Mode, it comes to return the setting value.

Remarks

Please use it after performing DioInit first and setting up an I/O port
by DioSet8255Mode() next, when using this function.

Example

C      Ret = DioGet8255Mode(Id, ChipNo, &CtrlWord);
