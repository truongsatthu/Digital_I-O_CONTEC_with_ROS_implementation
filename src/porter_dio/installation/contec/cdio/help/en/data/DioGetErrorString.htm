DioGetErrorString

Function

Retrieves the error string from error code.

Format

Ret = DioGetErrorString(ErrorCode , ErrorString);

Parameters

ErrorCode [C:long]
Specifies the return value of each function.

ErrorString [C:char *]
Sets the base address of buffer for storing error string.
The max length of string is 256.

Return Values

Ret [C:long]

Define Value Description
DIO_ERR_SUCCESS0Normal Complete
DIO_ERR_DLL_BUFF_ADDRESS10100Invalid data buffer address.
The other errors: (See also: Error code details)

Initial Values

None

Remarks

Utility function that returns the string in explicit description from error code.
Please use the error process.

Example

C       Ret = DioGetErrorString(ErrorCode, &ErrorString[0]);
