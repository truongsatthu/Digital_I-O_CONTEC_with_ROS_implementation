Uninstallation procedure

Uninstall by using uninstall shell script.
Please take care that the uninstall script doesn't stop the driver.
Perform uninstall with root authority.

 # cd contec/cdio
 # ./cdio_uninstall.sh
 ...................
 #

Please refer to cdio_uninstall.sh script to see the processing details. When the Makefile of Installer and shared library is rewritten, you must modify the contents of cdio_uninstall.sh script. Please note it.
