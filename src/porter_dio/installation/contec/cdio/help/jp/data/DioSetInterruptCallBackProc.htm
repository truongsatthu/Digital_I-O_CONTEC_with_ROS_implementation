DioSetInterruptCallBackProc

■機能

割り込みイベントコールバック関数を指定します。

■書式

Ret = DioSetInterruptCallBackProc(Id, CallBack, Param);

■引数

Id [C:short]
DioInitで取得したデバイスIDを指定します。

CallBack [C:PDIO_INT_CALLBACK]
コールバック関数のアドレスを指定します。
コールバック関数は関数ポインタで指定しますが、指定方法は言語により 異なりますので、注意してください。

Param [C:void *]
コールバック関数に渡したい任意のパラメータを指定します。

■戻り値

Ret [C:long]

定義 値 意味
DIO_ERR_SUCCESS0正常終了
DIO_ERR_DLL_INVALID_ID10001無効なIDが指定されました。
DIO_ERR_DLL_CALLBACK10400コールバック関数のアドレスが無効です。
その他のエラー：（参照：エラーコード詳細）

■説明

割り込み機能は、ハードウェア機能を使用して、指定したビットの入力値の変化を監視する機能です。
ビットの入力値が0から1、もしくは1から0になったとき、コールバックを通じてアプリケーションに 変化があったことを通知します。
割り込みコールバック設定を行った後、割り込みイベント設定を行い、割り込みが発生すると、 設定したコールバック関数が呼び出されます。

■使用例

C       Ret = DioSetInterruptCallBackProc(Id, CallBack, Param);
