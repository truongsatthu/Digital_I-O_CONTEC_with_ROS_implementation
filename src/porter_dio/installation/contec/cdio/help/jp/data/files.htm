  /User Directory/contec/cdio

    readme_jp.txt .................... README 日本語版

        Makefile ..................... コンパイル及びインストール用メイクファイル

        cgetres_usb.h ................ config用USBリソース取得 モジュールヘッダ

    /config
        config ....................... 設定プログラム
        contec_dio_start.sh .......... ドライバ起動スクリプト：configの実行により生成されます。
        contec_dio_stop.sh ........... ドライバ停止スクリプト：configの実行により生成されます。

        contec_dio.conf .............. 設定ファイル：configの実行により生成され、
                                       起動スクリプトにより/procにコピーされます。

        cdio.ko ...................... 起動するバージョンのドライバ
        cdiousb.ko ................... 起動するバージョンのドライバ

        /xxxx ........................ 現在のカーネルバージョンで命名するディレクトリ：メイクファイルの実行よりここにコピーされます。
            cgetres.ko ............... config用PCI, PCI Expressリソース取得ドライバ(現在のカーネルバージョンに対応)  
            cgetres_usb.ko ........... config用USBリソース取得ドライバ(現在のカーネルバージョンに対応)  
        /i386
            config ................... 32ビット設定プログラム               
        /amd64
            config ................... 64ビット設定プログラム

        cgetres.c .................... config用PCI, PCI Expressリソース取得 モジュールソース
        cgetres.h .................... config用PCI, PCI Expressリソース取得 モジュールヘッダ
