エラー一覧

　ステータス取得関数（DioDmGetStatus）で取得されるエラーの説明をします。
エラーは、以下に示す値の論理和で表されます。

定義 値 入出力 種類 内容
DIODM_STATUS_FIFOEMPTY00000001h出力のみエラー出力でFIFOが空になったことを示します。
主原因は、システムの負荷が高く、バスマスタ転送が間に合わないためです。
転送レートを下げる、システム負荷を軽くするなどの対策が必要です。
DIODM_STATUS_FIFOFULL00000002h入力のみエラー入力でFIFOが一杯になったことを示します。
主原因は、システムの負荷が高く、バスマスタ転送が間に合わないためです。
転送レートを下げる、システム負荷を軽くするなどの対策が必要です。
DIODM_STATUS_SGOVERIN00000004h入力のみエラーバッファがオーバーフローしたことを示します。
転送しようとしたデータ数がバッファサイズを超えています。
バッファサイズを大きくしてください。
DIODM_STATUS_TRGERR00000008h入力／出力エラー外部スタートで、スタート信号とストップ信号が同時に入ったことを示します。
このステータスが立った場合、転送は行なわれません。
外部スタート信号と外部ストップ信号の入り方を確認してください。
DIODM_STATUS_CLKERR00000010h入力／出力エラー外部クロックで、データ入出力中に次のクロックが入ったことを示します。
ボードの動作速度上、このステータスが立つことは、まずありません。
もしこのステータスが立った場合、外部クロックを下げることを検討してください。
