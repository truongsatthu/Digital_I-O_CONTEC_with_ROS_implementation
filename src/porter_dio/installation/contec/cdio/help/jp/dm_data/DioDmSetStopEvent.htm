DioDmSetStopEvent

■機能

転送完了のメッセージを通知するように設定します。

■書式

Ret =DioDmSetStopEvent(Id, Dir, CallBack, Param);

■引数

Id [C:short];
DioInitで取得したデバイスIDを指定します。

Dir [C:unsigned long]
転送方向を設定してください。

定義 値 意味
DIODM_DIR_IN0x01入力
DIODM_DIR_OUT0x02出力

CallBack [C:PDIO_DM_STOP_CALLBACK]
コールバック関数のアドレスを指定します。

Param [C:void *]
コールバック関数に渡したい任意のパラメータを指定します。

■戻り値

Ret [C:long]

定義 値 意味
DIO_ERR_SUCCESS0正常終了
DIO_ERR_DLL_INVALID_ID10001無効なIDが指定されました。
DIO_ERR_DLL_CALL_DRIVER10002ドライバを呼び出せません(ioctlに失敗)。
DIO_ERR_DLL_BUFF_ADDRESS10100データバッファアドレスが不正です。
DIO_ERR_DLL_CALLBACK10400コールバック関数のアドレスが無効です。
DIO_ERR_SYS_DIRECTION50000入出力方向が設定範囲外です。
その他のエラー：（参照：エラーコード詳細）

■初期値

なし

■説明

この関数を使用すると、サンプリング／ジェネレーティングの完了時にコールバック関数で完了を通知させることができます。

■使用例

C       Ret =DioDmSetStopEvent(Id, DIODM_DIR_IN, CallBack, Param);

■関連項目

PDIO_DM_STOP_CALLBACK
