DioGetErrorString

■機能

エラーコードからエラー文字列を取得します。

■書式

Ret = DioGetErrorString(ErrorCode , ErrorString);

■引数

ErrorCode [C:long]
各機能関数の戻り値を指定します。

ErrorString [C:char *]
エラー文字列を格納するバッファの先頭アドレスを指定します。
文字列の長さは、最大256文字です。

■戻り値

Ret [C:long]

定義 値 意味
DIO_ERR_SUCCESS0正常終了
DIO_ERR_DLL_BUFF_ADDRESS10100データバッファアドレスが不正です。
その他のエラー：（参照：エラーコード詳細）

■初期値

なし

■説明

エラーコードから、意味のわかる文字列を返すユーティリティ関数です。
エラー処理にご使用ください。

■使用例

C       Ret = DioGetErrorString(ErrorCode, &ErrorString[0]);

■関連項目

なし
