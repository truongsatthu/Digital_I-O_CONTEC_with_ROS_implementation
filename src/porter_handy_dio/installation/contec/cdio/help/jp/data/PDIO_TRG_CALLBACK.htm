PDIO_TRG_CALLBACK

■機能

トリガコールバックの形式です。

■書式

CallBackProc(Id, Message, wParam, lParam, Param);

■引数

Id [C:short]
DioInitで取得したデバイスIDが渡されます。

Message [C：short]
コールバック関数呼び出しの原因となるメッセージ番号が渡されます。

定義 値
DIOM_TRIGGER1340H

wParam [C:long]
トリガビット番号が渡されます。

lParam [C:long]
トリガ論理が渡されます。

定義 値 意味
DIO_TRG_RISE1トリガ：入力値0->1
DIO_TRG_FALL2トリガ：入力値1->0

Param [C:void *]
DioSetTrgCallBackProcで渡した任意の引数が渡されます。

■戻り値

なし

■説明

トリガコールバック関数の実体定義は、この書式で行ってください。

■関連項目

DioSetTrgEvent
