PDIO_INT_CALLBACK

■機能

割り込みコールバックの形式です。

■書式

CallBackProc(Id, Message, wParam, lParam, Param);

■引数

Id [C:short]
DioInitで取得したデバイスIDが渡されます。

Message [C：short]
コールバック関数呼び出しの原因となるメッセージ番号が渡されます。

定義 値
DIOM_INTERRUPT1300H

wParam [C:long]
割り込みビット番号が渡されます。

lParam [C:long]
割り込み論理が渡されます。

定義 値 意味
DIO_INT_RISE1割り込み：入力値0->1
DIO_INT_FALL2割り込み：入力値1->0

Param [C:void *]
DioSetInterruptCallBackProcで渡した任意の引数が渡されます。

■戻り値

なし

■説明

割り込みコールバック関数の実体定義は、この書式で行ってください。

■関連項目

DioSetInterruptEvent
