PDIO_DM_COUNT_CALLBACK

■機能

指定個数転送完了コールバックの形式です。

■書式

CallBackProc(Id, Message, Dir, Param);

■引数

Id [C:short]
DioInitで取得したデバイスIDが渡されます。

Message [C：short]
コールバック関数呼び出しの原因となるメッセージ番号が渡されます。

定義 値
DIO_DMM_STOP1440H

Dir [C:unsigned long]
転送方向が渡されます。

定義 値 意味
DIODM_DIR_IN0x01入力
DIODM_DIR_OUT0x02出力

Param [C:void *]
DioDmSetCountEventで渡した任意の引数が渡されます。

■戻り値

なし

■説明

転送完了コールバック関数の実体定義は、この書式で行ってください。

■関連項目

DioDmSetCountEvent
