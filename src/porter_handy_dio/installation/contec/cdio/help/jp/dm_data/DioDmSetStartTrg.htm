DioDmSetStartTrg

■機能

サンプリング、ジェネレーティングのスタート条件を設定します。

■書式

Ret =DioDmSetStartTrg(Id, Dir, Start);

■引数

Id [C:short];
DioInitで取得したデバイスIDを指定します。

Dir [C:unsigned long]
転送方向を設定してください。

定義 値 意味
DIODM_DIR_IN0x01入力
DIODM_DIR_OUT0x02出力

Start [C:unsigned long]
スタート条件を設定してください。

定義 値 意味
DIODM_START_SOFT1ソフトスタート
DIODM_START_EXT_RISE2外部トリガ立ち上がり
DIODM_START_EXT_FALL3外部トリガ立ち下り
DIODM_START_PATTERN4パターン一致(入力のみ）
DIODM_START_EXTSIG_15SCコネクタEXTSIG1
DIODM_START_EXTSIG_26SCコネクタEXTSIG2
DIODM_START_EXTSIG_37SCコネクタEXTSIG3

EXTSIG1〜EXTSIG3はスレーブ時のみ有効です。

■戻り値

Ret [C:long]

定義 値 意味
DIO_ERR_SUCCESS0正常終了
DIO_ERR_DLL_INVALID_ID10001無効なIDが指定されました。
DIO_ERR_DLL_CALL_DRIVER10002ドライバを呼び出せません(ioctlに失敗)。
DIO_ERR_SYS_DIRECTION50000入出力方向が設定範囲外です。
DIO_ERR_SYS_START50002使用できるスタート条件の範囲外です。
その他のエラー：（参照：エラーコード詳細）

■初期値

デフォルトはソフトスタートです。

■説明

アプリケーションの動作を一定にするために、スタート条件は必ず設定してください。
スタート条件にパターン一致を指定した場合、DioDmSetStartPattern関数で比較パターンを設定してください。

■使用例

C       Ret =DioDmSetStartTrg(Id, DIODM_DIR_IN, DIODM_START_SOFT);

■関連項目
