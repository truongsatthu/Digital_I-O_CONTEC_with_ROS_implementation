DioDmStart

■機能

サンプリング／ジェネレーティングを開始します。

■書式

Ret =DioDmStart(Id, Dir);

■引数

Id [C:short];
DioInitで取得したデバイスIDを指定します。

Dir [C:unsigned long]
転送方向を設定してください。

定義 値 意味
DIODM_DIR_IN0x01入力
DIODM_DIR_OUT0x02出力

■戻り値

Ret [C:long]

定義 値 意味
DIO_ERR_SUCCESS0正常終了
DIO_ERR_DLL_INVALID_ID10001無効なIDが指定されました。
DIO_ERR_DLL_CALL_DRIVER10002ドライバを呼び出せません(ioctlに失敗)。
DIO_ERR_SYS_DIRECTION50000入出力方向が設定範囲外です。
その他のエラー：（参照：エラーコード詳細）

■初期値

なし

■説明

この関数により、スタート条件がソフトスタートの場合ただちに、他の条件の場合はスタート条件が成立した時点で転送がスタートします。
転送スタートは、入力と出力を同時に指定できます。
複数枚のボードを同期させて使用したい場合、スレーブ側から先にスタートしてください。

■使用例

C       Ret =DioDmStart(Id, DIODM_DIR_IN);

■関連項目

なし
