チュートリアル

API-DIO(LNX)を使用してアプリケーションを作成するには、言語毎に以下のようにして 作成します。

ヘッダーファイル、pasファイルは編集しないでください。

■gcc


・C言語のヘッダーファイルcdio/samples/inc/cdio.hを作成するプログラムの入った ディレクトリにコピーしてください。
・ソースコード中に下記のように記述して、ヘッダーファイルをインクルードしてください。
#include "cdio.h"
・コンパイル時に、以下のようなリンカオプションを付けてリンクしてください。
gcc -Wall filename.c -o filename -lcdio
                                 ~~~~~~

詳細はサンプルプログラムのソースコード、Makefileなどを参照してください。
