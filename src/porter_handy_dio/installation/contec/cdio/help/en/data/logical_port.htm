Logical Port

The configuration of I/O ports for input and output of boards is different according to board.
For example: Physical Port
PIO-32/32L(PCI): Input:port +0 ~ port +3, Output:port +4 ~ port +7
PIO-64/64L(PCI): Input:port +0 ~ port +7, Output:port +8 ~ port +15

As for logical port, you can program without care about the configuration I/O address for each board, input and output from port 0 as the first number.
When API-DIO(LNX) is used, this logical port is used to input and output.
If the above-mentioned example is expressed as logical ports, it is like that below.

For example: Logical Port
PIO-32/32L(PCI): Input:port 0 ~ port 3, Output:port 0 ~ port 3
PIO-64/64L(PCI): Input:port 0 ~ port 7, Output:port 0 ~ port 7
