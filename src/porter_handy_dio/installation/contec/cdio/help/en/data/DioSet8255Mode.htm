DioSet8255Mode

Function

8255 mode setup is set up per chip.

Format

Ret = DioSet8255Mode(Id, ChipNo, CtrlWord);

Parameters

Id [C:short]
Specifies the device ID retrieved from DioInit.

ChipNo [C:unsigned short]
Specify a LSIi8255 chip number.

CtrlWord [C:unsigned short]
Specify control word.
The setting value list of control word is shown below.

Initial Values

At the time of system starting, all ports are setup of an input.

Remarks

Please use an I/O function after performing DioInit first and
setting up an I/O port with this function next, when using PIO-48D(PCI), PIO-48D(CB)H, DIO-48DX-USB.

Example

C      Ret = DioSet8255Mode(Id, ChipNo, CtrlWord);
