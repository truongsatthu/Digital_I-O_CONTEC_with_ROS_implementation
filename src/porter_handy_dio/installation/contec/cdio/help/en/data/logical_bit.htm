Logical Bit

The notation of I/O bits for input and output of boards is different according to board.
For example: Bit Notation
PIO-32/32L(PCI): Input:I00 ~ I37, Output:O40 ~ O77
PIO-64/64L(PCI): Input:I00 ~ I77, Output:O80 ~ OF7

As for logical port, you can program without care about the configuration I/O address for each board, input and output from bit 0 as the first number.
When API-DIO(LNX) is used, this logical bit is used to input and output.
If the above-mentioned example is expressed as logical bits, it is like that below.

For example: Logical Bit
PIO-32/32L(PCI): Input:0 ~ 31, Output:0 ~ 31
PIO-64/64L(PCI): Input:0 ~ 63, Output:0 ~ 63
