Trigger Function

Trigger function is a function that uses the OS timer to monitor the change of the input bit of digital input and output board. When the value changes, the application is notified.

The cycle and accuracy of monitoring trigger are controlled by OS function.

Trigger function is used, so application need not have the function of bit monitoring, comparison logic and so on.

When the different bits change at same time in monitoring cycle, the change is notified in turn from low bit(logical bit 0).
