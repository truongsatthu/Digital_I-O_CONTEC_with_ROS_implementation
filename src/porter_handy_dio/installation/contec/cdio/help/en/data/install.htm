Installation procedure

This driver is provided in a compressed file. You can use the following command in shell to copy/extract it.

When the module is compiled, Linux kernel's source tree is necessary. Beforehand, open the source code from the distribution CD.

If you want to change the library install path to a location, such as /usr/local/lib, you should find the Makefile and change the install path in advance.

-XXX is driver's version.

It is required to startup with root authority for install.

1)Copy the compressed file and extract it.

 # cd
 # mount /dev/cdrom /mnt/cdrom
 # cp /mnt/cdrom/linux/dio/cdioXXX.tgz ./
 # tar xvfz cdioXXX.tgz
 ................


2)Execute make, prepare to install the driver.

 # cd contec/cdio
 # make
 ................
 # make install
 ................


3)Setup the being used board by using setup tool (configuration).

 # cd config
 # ./config
 ..... Setup the following .........


4)Start the driver by start script of driver.

 # ./contec_dio_start.sh

Start and stop the driver by executing contec_dio_start.sh and contec_dio_stop.sh.
If you want to start driver when the system started, please add the processing contents of start script to /etc/rc.d/rc.local.
