Feature

- API-DIO(LNX) supplies the set of functions for controlling our digital I/O board by the driver and the shared library in module.

- The function is supported that is pattern input and output used by BusMaster transfer.

- When BusMaster transfer is not used, it can be used as a normal PIO board.

- The basic functionality of setting mode, input, output, interrupt and monitoring trigger based on timer is provided.

- Setup and use the device to be used by using the setup program (config) and setup files.

- Setup program generates the setting file, driver start script file required for shifting to executing environment and stop script file.

- The source code of user interrupt processing which can execute with the driver is supplied.
