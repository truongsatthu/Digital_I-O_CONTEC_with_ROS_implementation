Tutorial

Create the application by using API-DIO(LNX) as follows.

Don't edit the header file.

gcc


- Copy the header file of C language(cdio/samples/inc/cdio.h) to the directory of creating application.
- Add the following to the source code, include the header file.
#include "cdio.h"
- When compiling, please specify linker options shown as below to link.
gcc -Wall filename.c -o filename -lcdio
                                 ~~~~~~

For details, refer to the source code of sample program and Makefile.
