DioDmStop

Function

Stops sampling/generating.

Format

Ret =DioDmStop(Id, Dir);

Parameters

Id [C:short]
Specifies the device ID retrieved from DioInit.

Dir [C:unsigned long]
Specify transfer direction.

Define Value Description
DIODM_DIR_IN0x01Input
DIODM_DIR_OUT0x02Output

Return values

Ret [C:long]

Define Value Description
DIO_ERR_SUCCESS0Normal Complete
DIO_ERR_DLL_INVALID_ID10001Invalid ID is specified.
DIO_ERR_DLL_CALL_DRIVER10002Driver cannot be called (failed in ioctl).
DIO_ERR_SYS_DIRECTION50000I/O direction is outside of the setting range.
The other errors: (See also: Error code details)

Initial value

None

Remarks

With this function, transfers are compulsorily stopped (software stop) even though stop conditions are not met.
Input and output can be specified simultaneously for transfer stop.

Example

C       Ret =DioDmStop(Id, DIODM_DIR_IN);
