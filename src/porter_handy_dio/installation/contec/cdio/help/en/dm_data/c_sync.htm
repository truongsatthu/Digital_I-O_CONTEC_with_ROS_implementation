Console: sync

This is a program that executes the synchronous sampling by using synchronization control connector.
Input a number of the menu list for selecting the menu.
After executes Master and Slave setting,
transfer is started in order of Slave -> Master.
Sampling data stored in current buffer would be saved by selecting "Data" menu.

*Notice

Please do not change a terminal in size with the sample which used the curses library.
A hang-up or a segmentation error may be caused under the influence of a curses library.
